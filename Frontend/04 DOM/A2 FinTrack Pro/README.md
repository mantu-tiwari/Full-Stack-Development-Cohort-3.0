# 💰 FinTrack Pro

A modern and responsive **Personal Finance Tracker** built using **HTML, CSS, and JavaScript**. FinTrack Pro helps users manage income and expenses, visualize financial data with charts, and securely store transactions using LocalStorage.

## 📸 Preview

![FinTrack Pro Screenshot](./FinTrack-Pro%20Dashboard.png)

---

## 🚀 Live Demo

🔗 **Live Website:** https://fintrack-pro-mantu-tiwari.vercel.app/

---

## ✨ Features

- 📊 Interactive financial dashboard
- 💵 Track Income and Expenses
- 💰 Real-time Current Balance calculation
- 📈 Cash Flow Analysis using Chart.js
- 📝 Add, Edit & Delete transactions
- 🔍 Search transactions instantly
- 🏷️ Filter transactions by category
- 🌙 Dark Mode support
- 💾 LocalStorage support (data persists after refresh)
- 🔄 Reset all data with confirmation

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/mantu-tiwari/fintrack-pro.git
```

Go to the project folder

```bash
cd fintrack-pro
```

Open `index.html` in your browser.
